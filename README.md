ambulance-webapp
================

Automation of medical establishment
+ authorization of doctors
+ receiving patients
+ referral of patients for blood test
+ laboratory module
+ accounting of sicknesses

Used technologies:
+ Spring 3.1.0 (Core, Security)
+ Hibernate 3.6
+ Tomcat 7.0.3
+ JSF 2
+ UI: Primefaces 3.4.2
+ PostgreSQL 9.x
+ Maven
